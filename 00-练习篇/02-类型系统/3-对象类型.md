# 3-对象类型

> 知识大纲

* 内置对象类型
    * 在js中，有许多的内置对象，比如：Object，Array，Date等，我们可以通过对象的构造函数或者类来进行标注
        ```ts
        let a: Object = {};
        let arr: Array<number> = [1, 2, 3];
        let d: Date = new Date();
        ```

* 自定义对象类型
    * 另外一种情况，许多时候，我们可能需要自定义结构的对象，这个时候我们可以：
        * 字面量标注
            ```ts
            let a: {username: string, age: number} = {
                username: 'gqf',
                age: 28
            }
            // ok
            a.username
            a.age
            // error
            a.gender
            ```
        * 接口
        * 定义类或者构造函数
